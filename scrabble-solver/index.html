Scrabble solver

During my last year in Licence I implemented a Scrabble solver in Python. I implemented the World's Fastest Scrabble algorithm. This algorithm find the best word according to the letters on the board game and in the player's hand.

First I needed to build a trie with all the possible words. Then the algorithm go through this trie using the letters in hand and already on the board game to find the word wich cost the most point.

This application is developped in Python 3. I implemented an user interface with the Tkinter library. You can find the source code on my GitHub repository.
